Why should a mobile app design team focus on a wireframe before a prototype?

    A wireframe is a low-fidelity representation of an app's layout that all team members can reference and critique before focusing on visual design and user interactions. 

        Sketching out a wireframe can sace you and your team hours of work by understanding the basic skeleton of what you're building. 

---

What is the primary purpose of a prototype?

    A prototype is meant to serve as a bridge between the user and the actual product so that designers can test how users interact with the interface. 

        Prototypes allow for testing user flow through an app or website.

Which of the following is NOT a valid method of prototyping?

    Graphic design software

        Graphic design software can be used to create wireframes, but it is not an ideal tool for creating prototypes.
