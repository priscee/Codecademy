Which of the following will set the right padding to 10 pixels, bottom padding to 15 pixels, left to 20 pixels, and top padding to 25 pixels?

    padding: 25px 10px 15px 20px

        When four values are provided to padding(or margin), they are applied in the order: top, right, bottom, left.

In the sample code below, the element with ID “one” is directly above the element with ID “two” in the HTML document. How much space is between the border of element #one and the border of element #two?

#one {
  margin: 20px;
}

#two {
  margin: 30px;
}

    30px

        The vertical margins of these two elements will not add together but will collapse to the larger value of the two, 30 pixels.

Assuming the .box element below is the only element on the page, what is the total height calculated by using the given CSS properties?

.box {  
  border: 1px solid black;
  padding: 10px;
  height: 125px;
}

    147px

        The total element height inclues the height property, the top and bottom border heights, and the top and bottom padding heights (125 +10 +10 +1 + 1)
