What does z-index control?

    How far 'back' or 'forward' overlapping elements appear on a page. 

        z-index controls the relative depth of overlapping elements.

Complete the statement: An inline-block element . . .

    does not start a new line and can be sized using the height and width properties. 

        inline-block elements can flow on the same line, and their dimensions can be set with height and width.

How will this rule set affect the .title-card element?

.title-card {
  position: fixed;
  top: 0px;
}

    Remove .title-card from the flow of the document and keep it in the same position within the browser window, even if the user scrolls. 

        position: fixed is used to fix elements to a position on the page regardless of scrolling.
